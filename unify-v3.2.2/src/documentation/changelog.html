v3.0 - 06 December, 2021

Unify v3.0 is a major rewrite of the entire template with new a design system. Fully re-designed from scratch and built on top of the latest Bootstrap v5. Hence, the migration from v2.x to v3.x is not possible. Below are most notable changes made in Unify v3.0.

  • Bootstrap v5 - Built on top of the latest Bootstrap v5 from scratch.
  • JavaScript - All scripts are fully rewritten from scratch in pure JavaScript without jQuery.
  • Design - All pages are designed from scratch with a new fresh look and design language.
  • Documentation - New complete "Documentation" for all components with detailed information.
  • Snippets - Fully redesigned "Snippets" pages to easily build new pages.
